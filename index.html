Introduction

Le projet YouGo App s'inscrit dans le cadre du module Projet Intégrateur de l'année académique 2024–2025 à l'Institut de Formation et de Recherche en Informatique (IFRI), Université d'Abomey-Calavi.

Dans un contexte marqué par la croissance des enjeux de mobilité urbaine, ce projet propose une solution numérique concrète de covoiturage communautaire intelligent visant à connecter les étudiants entre eux pour optimiser leurs déplacements quotidiens vers le campus.

🎯 Problématique Centrale

La problématique abordée consiste à faciliter les trajets partagés entre étudiants à travers une plateforme intuitive, sécurisée et géolocalisée. Elle intègre une messagerie en temps réel, un système de matching conducteur-passager, un suivi GPS et bien d'autres services innovants.

💡 Objectif Technique

Mobiliser les compétences techniques acquises en algorithmique, développement web, base de données et architecture logicielle.

🤝 Objectif Collaboratif

Développer l'esprit d'équipe et la collaboration agile entre étudiants de différentes filières informatiques.

📋 Structure du Rapport

Ce rapport présente de manière structurée l'ensemble de la démarche suivie, depuis l'analyse des besoins jusqu'à la réalisation fonctionnelle de l'application YouGo, en passant par la répartition des tâches, la conception technique et les solutions mises en œuvre.

Mode de Fonctionnement Interne

🎯 Méthodologie de Travail

Notre équipe a adopté une approche Agile structurée en sprints, avec une répartition claire des responsabilités selon les spécialités de chaque membre. Cette méthode nous a permis de développer YouGo de manière itérative et collaborative, en garantissant une qualité constante et une livraison régulière de fonctionnalités.

📋 Organisation en Sprints - Portfolio Technique Détaillé

Le développement de YouGo s'est articulé autour de 4 sprints majeurs, chacun ciblant des fonctionnalités spécifiques avec une répartition équilibrée entre développement frontend et backend (structuration_you_go.pdf). Cette approche a permis une intégration continue et un suivi précis des performances de chaque membre de l'équipe.

Instructions de Déploiement

📋 Prérequis

  • Python 3.9+ installé sur le système
  • MySQL 8.0+ configuré et accessible
  • Redis pour les WebSockets et cache
  • Git pour cloner le repository

⚙️ Installation

1. Cloner le repository

git clone https://github.com/[USERNAME]/PIL1_2425_[NUMERO_GROUPE].git
cd PIL1_2425_[NUMERO_GROUPE]

2. Créer l'environnement virtuel

python -m venv venv
source venv/bin/activate  # Linux/Mac
# ou
venv\Scripts\activate     # Windows

3. Installer les dépendances

pip install -r requirements.txt

4. Configuration de l'environnement

Créer un fichier .env à la racine du projet :

DJANGO_SECRET_KEY=votre_clé_secrète_très_sécurisée
DEBUG=True
DB_NAME=ifri_comotorage_db
DB_USER=root
DB_PASSWORD=votre_mot_de_passe_mysql
DB_HOST=localhost
DB_PORT=3306
EMAIL_HOST_USER=note.yougo@outlook.fr
EMAIL_HOST_PASSWORD=votre_mot_de_passe_email
REDIS_URL=redis://localhost:6379/0

5. Configuration de la base de données

# Créer la base de données MySQL
mysql -u root -p
CREATE DATABASE ifri_comotorage_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit

# Appliquer les migrations Django
python manage.py makemigrations
python manage.py migrate

6. Créer un superutilisateur

python manage.py createsuperuser

7. Lancer le serveur de développement

# Terminal 1 - Serveur Django
python manage.py runserver

# Terminal 2 - Worker Celery (optionnel)
celery -A core worker --loglevel=info

# Terminal 3 - Redis (si pas installé comme service)
redis-server

🌐 Accès à l'application

  • Application principale : http://localhost:8000
  • Interface Admin : http://localhost:8000/admin
  • API Documentation : http://localhost:8000/api/docs

Manuel d'Utilisation

🏁 Premiers pas

1. Inscription et Connexion

  • Accéder à la page d'inscription via le bouton "S'inscrire"
  • Remplir les informations : nom, prénom, email, téléphone, mot de passe
  • Choisir votre rôle principal : Conducteur ou Passager
  • Valider votre compte via l'email de confirmation reçu
  • Se connecter avec email/téléphone + mot de passe

2. Configuration du Profil

  • Compléter votre profil avec photo et informations personnelles
  • Définir votre point de départ habituel (adresse ou géolocalisation)
  • Configurer vos horaires de trajet habituels
  • Si conducteur : ajouter les informations de votre véhicule
  • Activer les notifications selon vos préférences

🚗 Utilisation - Conducteur

Publier une Offre de Covoiturage

  1. Cliquer sur "Proposer un trajet"
  2. Définir le point de départ et d'arrivée
  3. Préciser l'heure de départ et le nombre de places disponibles
  4. Fixer le prix par passager (suggestion automatique basée sur la distance)
  5. Ajouter des notes spécifiques (préférences, points d'arrêt, etc.)
  6. Publier l'offre - elle sera visible dans les recherches

Gérer vos Trajets

  • Consulter les demandes reçues dans l'onglet "Mes demandes"
  • Accepter ou refuser les passagers selon vos critères
  • Communiquer via la messagerie intégrée
  • Démarrer le suivi GPS une fois le trajet commencé
  • Finaliser le trajet et confirmer la facturation

🎒 Utilisation - Passager

Rechercher un Trajet

  1. Utiliser la fonction "Rechercher un trajet"
  2. Saisir votre point de départ et destination
  3. Sélectionner la date et heure souhaitées
  4. Parcourir les offres disponibles avec filtrages
  5. Consulter les profils des conducteurs et leurs évaluations

Réserver et Voyager

  • Envoyer une demande de réservation au conducteur
  • Attendre la confirmation et payer en ligne si requis
  • Communiquer avec le conducteur via le chat
  • Suivre la position du conducteur en temps réel
  • Évaluer le trajet une fois terminé

💬 Messagerie et Communication

  • Chat temps réel avec notifications instantanées
  • Partage de localisation en direct
  • Messages vocaux pour plus de simplicité
  • Historique des conversations sauvegardé
  • Support client intégré via chatbot

📊 Tableau de Bord Personnel

  • Statistiques de vos trajets (kilomètres, économies, CO2)
  • Historique complet avec possibilité d'export
  • Gestion des factures et paiements
  • Badges de fidélité et récompenses
  • Évaluations reçues et données de fiabilité
